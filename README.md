# 🍳 MealSpark

[![Live Demo](https://img.shields.io/badge/Live_Demo-mealspark.tech-success?style=for-the-badge)](https://mealspark.tech)
[![Deployed on Vercel](https://img.shields.io/badge/Vercel-Deployed-black?style=for-the-badge&logo=vercel)](#)

MealSpark is an AI-powered web app designed to cure "fridge blindness." Snap a photo of the ingredients you have, and it instantly crafts a custom recipe to turn what is in your fridge into an actual meal.

![MealSpark Demo](demo.gif)

## ✨ How It Works
1. **Upload:** Drop in a photo of your available ingredients.
2. **Analyze:** The AI scans the image and identifies what you have.
3. **Cook:** Get instant step-by-step cooking instructions and measurements.

## 🚀 Try It Out
No accounts, downloads, or setup required. You can test the live version here: **[mealspark.tech](https://mealspark.tech)**

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** AI Integration API
* **Deployment:** Vercel
